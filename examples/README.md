# Examples

Runnable pure-Ruby usage of the `cgi` escaping and query-parsing surface, verified under the [rbgo](https://github.com/go-embedded-ruby/ruby) interpreter.

```sh
rbgo examples/cgi_usage.rb
```

| File | Shows |
| --- | --- |
| `cgi_usage.rb` | `CGI.escape`/`unescape`, `escapeURIComponent`/`unescapeURIComponent`, `escapeHTML`/`unescapeHTML`, `escapeElement`, and `CGI.parse` |
